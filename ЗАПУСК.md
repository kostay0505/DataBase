# Инструкция по запуску Notes App

## Что нужно установить заранее

1. **Docker Desktop** — главный инструмент для запуска всего проекта.
   Скачай с официального сайта: docker.com/products/docker-desktop
   После установки запусти Docker Desktop и убедись, что он работает (иконка в трее).

2. **Node.js** — нужен только если хочешь запускать фронт/бот вне Docker (опционально).
   Скачай с nodejs.org (версия 20 LTS).

---

## Шаг 1 — Создай `.env` файл

В папке `E:\DataBase\` создай файл `.env` (не `.env.example`, а именно `.env`):

```
BOT_TOKEN=сюда_вставишь_токен_бота
TRILIUM_TOKEN=сюда_вставишь_позже
```

Пока оставь `TRILIUM_TOKEN` пустым — получишь его на шаге 3.

---

## Шаг 2 — Запусти только Trilium

Открой терминал (PowerShell или CMD) в папке `E:\DataBase\` и выполни:

```bash
docker compose up trilium -d
```

Подожди ~30 секунд, затем открой в браузере:
**http://localhost:8080**

Trilium попросит создать пароль при первом запуске — создай и запомни его.

---

## Шаг 3 — Получи ETAPI токен

1. В Trilium (http://localhost:8080) нажми на иконку шестерёнки (Settings)
2. Найди вкладку **ETAPI**
3. Нажми **"Create token"**
4. Скопируй токен и вставь в `.env`:

```
TRILIUM_TOKEN=вставь_скопированный_токен_сюда
```

---

## Шаг 4 — Получи токен Telegram бота

1. Открой Telegram, найди **@BotFather**
2. Отправь `/newbot`
3. Придумай имя бота (например: `My Notes Bot`)
4. Придумай username (например: `my_notes_123_bot`) — должен заканчиваться на `bot`
5. BotFather выдаст токен вида `1234567890:ABCdef...`
6. Вставь в `.env`:

```
BOT_TOKEN=1234567890:ABCdef...
```

---

## Шаг 5 — Запусти весь проект

```bash
docker compose up -d
```

Docker соберёт образы (первый раз ~5 минут) и запустит все три сервиса.

---

## Шаг 6 — Проверь что всё работает

```bash
docker compose ps
```

Должно показать три контейнера со статусом `running`:

```
database-trilium-1    running   0.0.0.0:8080->8080/tcp
database-frontend-1   running   0.0.0.0:3000->3000/tcp
database-bot-1        running
```

Открой:
- **http://localhost:3000** — веб-интерфейс заметок
- **http://localhost:8080** — Trilium напрямую

В Telegram найди своего бота и отправь `/note Первая заметка` — заметка должна появиться на сайте.

---

## Полезные команды

```bash
docker compose logs -f          # логи всех сервисов
docker compose logs -f bot      # только логи бота
docker compose restart          # перезапустить всё
docker compose down             # остановить всё
docker compose up -d --build    # пересобрать и запустить (после изменений в коде)
```

---

## Если что-то не работает

| Проблема | Решение |
|---|---|
| Порт 8080 занят | Измени в `docker-compose.yml` на `"8081:8080"` |
| Порт 3000 занят | Измени на `"3001:3000"` |
| Бот не отвечает | Проверь `docker compose logs bot` — скорее всего неверный `BOT_TOKEN` |
| Фронт не видит заметки | Проверь что `TRILIUM_TOKEN` в `.env` правильный |
